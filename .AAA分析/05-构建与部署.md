# 构建与部署指南

## 构建系统

### 构建脚本

| 命令 | 说明 | 输出 |
|------|------|------|
| `npm run build` | tsup 构建 | dist/ |
| `npm run typecheck` | TypeScript 类型检查 | - |
| `npm run prepublishOnly` | 发布前构建 | dist/ |

### 构建配置 (tsup.config.ts)

```typescript
import { defineConfig } from 'tsup';

export default defineConfig({
  entry: ['src/index.ts'],
  format: ['esm', 'cjs'],
  dts: true,
  splitting: false,
  sourcemap: true,
  clean: true,
  target: 'es2022',
});
```

### 构建输出

```
dist/
├── index.js          # ESM 入口
├── index.d.ts        # ESM 类型声明
├── index.cjs         # CJS 入口
├── index.d.cts       # CJS 类型声明
├── index.js.map      # Source map
└── index.cjs.map     # Source map
```

## 开发环境

### 开发命令

| 命令 | 说明 |
|------|------|
| `tsx src/index.ts` | 直接执行入口（无需构建） |
| `npm run typecheck` | 类型检查 |
| `npm run build` | 构建 |

### 环境要求

- **Node.js**: >= 18
- **TypeScript**: >= 5.5
- **OpenCode**: >= 1.15.7

### 安装依赖

```bash
# 安装生产依赖
npm install sql.js uuid express cors compression

# 安装开发依赖
npm install -D typescript tsup tsx @opencode-ai/plugin

# 安装类型声明
npm install -D @types/node @types/uuid @types/express @types/cors @types/compression
```

## 插件安装

### 本地安装

1. 克隆或下载项目
2. 在 OpenCode 配置文件 `opencode.json` 中添加：

```json
{
  "plugin": ["file:./novel-weaver"]
}
```

### NPM 安装（发布后）

```bash
npm install novel-weaver
```

```json
{
  "plugin": ["novel-weaver"]
}
```

## 项目初始化

### 使用 novel_init 工具

```bash
novel_init project_name="轮回之塔" genre="infinite-flow"
```

### 初始化流程

```
1. 检查 .novel-weaver/ 目录是否存在
   │   ├── 存在: 返回错误（需手动删除）
   └── 不存在: 继续
   │
2. 创建目录结构
   │   ├── .novel-weaver/
   ├── .novel-weaver/content/
   ├── .novel-weaver/content/settings/
   ├── .novel-weaver/content/dungeons/
   ├── .novel-weaver/content/chapters/
   └── .novel-weaver/content/reports/
   │
3. 初始化数据库
   │   ├── 创建表
   ├── 创建索引
   ├── 创建 FTS4 虚拟表
   └── 执行迁移
   │
4. 生成核心世界观文件
   │
5. 完成初始化
```

## 数据存储

### 数据库文件

- **位置**: `.novel-weaver/novel-weaver.db`
- **引擎**: sql.js (WASM SQLite)
- **特性**: 同步操作、FTS4 全文搜索

### Markdown 文件

```
.novel-weaver/content/
├── settings/
│   ├── 核心世界观.md
│   ├── world-{name}.md
│   └── char-{name}.md
├── dungeons/
│   └── dungeon-{name}.md
├── chapters/
│   └── vol-N/
│       ├── ch{num}-{title}.md
│       └── ...
└── reports/
    ├── consistency-{date}.md
    └── progress-summary-{date}.md
```

### 向量存储

- **位置**: `.novel-weaver/vectors/`
- **格式**: JSON 文件
- **结构**: `{entityType}/{entityId}.json`

```json

  "genre": "infinite-flow",
  "author": "作者名",
  "temperature": {
    "plot-writer": 0.85,
    "reviewer": 0.25
  },
  "antiAi": {
    "enabled": true,
    "layers": ["adverb_overuse", "emotion_tagging"]
  },
  "dashboard": {
    "port": 3456,
    "host": "127.0.0.1"
  }
}
```

### 配置加载

```typescript
// src/tools/init.ts
function loadRcConfig(projectRoot: string): NovelWeaverRcConfig {
  const rcPath = path.join(projectRoot, '.novel-weaverrc.json');
  if (fs.existsSync(rcPath)) {
    return JSON.parse(fs.readFileSync(rcPath, 'utf-8'));
  }
  return {};
}
```

## 常见问题

### 构建失败

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| TypeScript 类型错误 | 类型不兼容 | 检查类型声明 |
| tsup 构建失败 | 配置错误 | 检查 tsup.config.ts |
| 缺少依赖 | 未安装依赖 | 运行 npm install |

### 运行时错误

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 数据库未初始化 | 未调用 novel_init | 先初始化项目 |
| sql.js 加载失败 | WASM 加载失败 | 检查 sql.js 安装 |
| 文件权限错误 | 权限不足 | 检查文件权限 |

### 插件加载失败

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| OpenCode 版本不兼容 | 版本过低 | 升级 OpenCode |
| 插件路径错误 | 路径配置错误 | 检查 opencode.json |
| 缺少 peer 依赖 | 未安装 @opencode-ai/plugin | 安装依赖 |
