## 核心特性

### 1. 无限流小说创作

专门针对无限流/副本流网文的创作辅助，提供：
- 核心世界 + 副本世界的层级设定
- 副本生成模板（恐怖、科幻、仙侠、都市、末世）
- 攻略进度追踪

### 2. 世界观管理

- 创建核心世界和副本世界
- 世界设定关联（wikilink）
- 实体关系图谱

### 3. 角色系统

- 角色创建和档案管理
- 角色状态追踪（按章节）
- 角色关系网络

### 4. 章节引擎

- 分场景调度写作
- 情感蓝图生成
- 类型模板支持（5种类型）
- 风格锚点提取

### 5. 质量审查

- 8 项标准检查
- AI 味检测（60+ 模式）
- 自动修复 blocker 问题

### 6. 一致性检查

- 跨世界设定一致性
- 角色状态一致性
- 自定义一致性规则

### 7. Pipeline 工作流

四阶段创作管线：
1. 设定阶段（世界观、角色、副本）
2. 规划阶段（章节大纲、剧情结构）
3. 写作阶段（章节正文）
4. 审查阶段（质量检查与修复）

### 8. Dashboard

- Web 可视化面板
- 项目状态监控
- 写作统计

### 9. RAG 检索

- 向量存储
- 嵌入生成
- 检索增强生成

## 构建系统

| 命令 | 说明 |
|------|------|
| `npm run build` | tsup 构建 (ESM + CJS + .d.ts) |
| `npm run typecheck` | TypeScript 类型检查 |

- `dist/index.js` - ESM 入口
- `dist/index.cjs` - CJS 入口

## 数据存储

### 数据库

- **引擎**: sql.js (WASM SQLite)
- **位置**: `.novel-weaver/novel-weaver.db`
- **特性**: 同步操作、FTS4 全文搜索

### 文件存储

```
.novel-weaver/
├── novel-weaver.db              # SQLite 数据库
└── content/
    ├── settings/                # 世界观 + 角色 .md
    ├── dungeons/                # 副本 .md
    ├── chapters/                # 章节 .md (按卷分目录)
    └── reports/                 # 报告 .md
```

### Obsidian 兼容

- YAML frontmatter
- Wikilink 引用 `[[实体名]]`
- 标签系统
- Graph View 关系图谱
